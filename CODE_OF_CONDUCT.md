# Code of conduct

Frostflare is meant to help friends understand and enjoy their communities. Treat contributors and users with patience, respect, and care for privacy.

Harassment, discrimination, threats, sexualized conduct, deliberate exposure of private data, and sustained disruption are not welcome. Project maintainers may remove contributions or participation that violate these expectations.

For security or privacy exposure, use [GitHub private vulnerability reporting](https://github.com/FrozenScorch/frostflare/security/advisories/new). For conduct concerns that are not security issues, contact the repository owner through their public GitHub profile without including sensitive details in a public issue.
